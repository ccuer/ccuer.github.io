Oracle 11g 下载及安装教程

2020-02-24
数据库
字数统计: 1.2k字   |   阅读时长≈ 4分钟

  1. 下载Oracle 11g R2 for Windows版本，官网的下载地址：
    Oracle 1of 2
    Oracle 2of 2
    但是亲测该服务器为美国服务器，下载速度简直不要太难受，因此附上网盘链接——Oracle百度网盘下载地址(点击自动跳转)
    https://pan.baidu.com/s/1BTnd8CtJrZfD3YAZI0_AQg，提取码：eder.
    如果有会员的同学可以欢快的下载啦，没有百度网盘会员的同学不要灰心，私聊博主，我给你们一个小诀窍，让你没有百度网盘网速的限制

  2. 百度网盘文件下载好以后，解压缩可以看到里面有这两个压缩包
    把这两个压缩包也解压出来

    然后打开这两个压缩包，就可以看到有一个相同的名为：database的文件夹
    将2of2中的database里的stage复制到1of2的database里面

    然后你就可以关掉2of2的文件夹了，点击打开1of2的database里的setup.exe

    注意：这里打开以后会出现一个类似于CMD命令窗口，不要关掉那个窗口，它很快就会自动关掉，这个时候需要稍微等一会，不要像我一样，以为是卡了，点了好多次，要记住，你点多少次，就是打开多少个安装程序

  3. 耐心等待之后是出现这样一个提示，告诉你，你的电脑可能太垃圾了，但是我们不用搭理它，直接点击 “是” 继续安装。

  1. 不选择更新，否则你好像需要购买口令

  1. 下一步之后到了让你选择安装方式的时候，由于我们之前没有装过Oracle，所以我们直接选择创建和配置数据库，否则你需要根据你的自身情况选择。下一步

  1. 接下来是选择你的安装环境，我们笔记本或者家用台式机一般使用的是Windows系统，所以直接选择桌面类即可；如果你是在服务器的Window
    Server系统上安装，请选择服务器类。    ，下一步

  2. 然后是安装配置页面,这里如果你不需要修改安装地址，那么可以像我一样直接看到管理口令这里，这个口令其实就是让你设置管理员密码。
    这里需要注意的是：管理员密码需要包含，至少一个大写字母、一个小写字母、一个数字、总共8位字符。
    否则会提示警告，正确输入以后就可以下一步了。

  3. 接下来这个地方，有点不一样，我一共装了两次，给笔记本装的时候是出现了一个先决条件检查的，但是同样的文件给台式机装就没有这个界面，如果你们出现了下面这个界面，可以点一下全部忽略，然后下一步，如果没有就算了。
    然后就是漫长的等待了。

  4. 在这个等待的过程中，可能会出现两种界面
    a.防火墙询问，就是问你让不让它访问互联网，不用想，肯定允许访问呐
    b.提示你需要下载安装一些windows功能，也是挨个点击下载并安装即可。
    注意，我的经验告诉我，这里会出现好多个窗口（估计我的至少有十个），简直多到不敢想象，让我有一种中病毒的既视感，给你们贴两张图感受感受


    当然，不排除我的电脑比较Low，你们的电脑功能比较多，不需要下载的情况，如果没出现这些窗口也不要害怕，直接往下操作即可。
    然后安装程序会自动再打开一个窗口如下（不用操作，耐心等待）：

  5. 经过very long long long 时间之后，我们迎来了新的窗口，在这个窗口，我们点击口令管理。

  6. 将sys和system的用户密码重新设置（可以和刚刚那个密码一样）
    将sh和scott账号的勾去掉 并重新设置密码
    输入完密码和确认密码之后，点击确定，就可以完成创建了

  7. 然后回到最初的安装界面，点击完成即可。
    这时，我们可以打开数据库，看看能否连接成功
    打开主菜单，找到Oracle->Database Control - orcl

  8. 打开之后就会跳转到网页登陆：在连接身份里选择”SYSDBA”,再用户名处输入”sys”，密码为你最初设定的密码，点击”登陆”你就可以访问数据库了


14. 点击登录就可以进入你的数据库实例了，是不是看着就很开心
