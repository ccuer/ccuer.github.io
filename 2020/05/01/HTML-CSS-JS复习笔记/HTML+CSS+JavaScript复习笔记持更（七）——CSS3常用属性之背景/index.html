背景设置

HTML页面中的背景是通过background设置的。

语法格式：

在背景设置中，有多种参数可选

  • background-color

  • background-image

  • background-repeat

  • background-attachment

  • background-position

    其中，color可以直接写颜色（red，green等），也可以写rgba值

    image是使用图片作为背景，使用语法：background-image: url(01.jpg);

repeat是设置背景图重复，通常有4种选项:repeat、no-repeat、repeat-x、repeat-y，分别意思是重复，不重复，在x轴上重复，在y轴重复

attachment是设置背景滚动的，有两个属性值：
     scroll：背景图片跟随滚动
     fixed：背景固定在可视区域（不随页面滚动）

position是设置背景位置,有7个属性值：
     length，设置背景图片与页面边距的水平和垂直方向的距离(单位：px、cm、mm等)
     percentage，根据页面的宽高百分比设置背景
     top，设置背景图片顶部居中显示
     center，设置背景图片居中显示
     bottom，设置背景图片底部居中显示
     left，设置背景图片左对齐
     right，设置背景图片右对齐

CSS3新属性

  • background-clip（指定背景的显示范围）
  • background-origin（指定背景图片的起点）
  • background-size（指定背景图片的大小）

background-clip

背景的显示范围。有三种取值：

     border-box（从border向外裁剪，不含border，即从border和外边距的交界开始，往外面没有背景）
     padding-box（从padding向外裁剪，不含padding，即从padding和外边距的交界开始，往外面没有背景）
     content-box（从内容区content往外裁剪）

background-size

背景的大小，有四种取值：

两个数值或者百分比：背景的宽、高（可以只给一个数值，即为宽，高度等比缩放）
auto：背景图的原始尺寸
cover：背景图拉伸缩放至容器大小，宽高都要达到大小，有可能图片超出容器
contain：背景图的宽或者高拉伸到容器大小，保证图片都在容器内
